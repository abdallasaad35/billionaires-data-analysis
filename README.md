# Billionaires Data Analysis

## 📊 Overview

This project analyzes a global dataset of billionaires to explore wealth distribution, industries, and country-based insights.

## 📁 Dataset

* File Name: Billionaires Statistics Dataset Raw.xlsx
* The dataset includes:

  * Rank
  * Person Name
  * Country & City
  * Industry / Category
  * Net Worth (finalWorth)
  * Self-made status
  * Gender
  * Economic indicators (GDP, population, tax rate, etc.)

## 🛠️ Tools Used

* Microsoft Excel

## 🔍 Data Cleaning

* Removed duplicate records
* Handled missing values
* Corrected data types
* Organized columns for better analysis

## 📈 Analysis & Insights

* Most billionaires are from the United States
* Technology and Finance sectors dominate
* A large percentage of billionaires are self-made

## 📊 Visualizations

Charts were created using Excel to represent:

* Distribution of billionaires by country
* Distribution by industry

## 🚀 How to Use

1. Open the dataset using Excel
2. Explore the cleaned data
3. Review the analysis and charts
